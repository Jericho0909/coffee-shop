## Coffee Shop Website:

A full-featured coffee shop web app built with React and styled using Tailwind CSS.  
It includes an Admin Dashboard for managing products, orders, employers, stocks, customers, and settings, as well as a Customer Interface for browsing and ordering coffee online.


## Features:

- View and order coffee items
- Firebase Authentication (Sign up, Login, Email verification)
- Admin Dashboard for managing products, orders, and employers
- JSON Server / Firebase backend
- Fully responsive design

## Tech Stack:

- Frontend:	React, Tailwind CSS, Framer Motion

- Backend:	JSON Server / Firebase

- Hosting:	Netlify

- Version:    Control	Git + GitHub

## Future Improvements:

- Add product rating and review system

- Add discount and promo features for selected coffee items 


## Clone the repository:

git clone https://github.com/Jericho0909/coffee-shop.git

cd coffee-shop

## Install dependencies:

npm install

## Run the app:

npm start
